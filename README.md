# Delta

Calendar-first planning app that converts brain dumps into actionable event/task proposals.

## Stack

- React + TypeScript + Vite
- Frontend-first scaffold
- Typed API stubs for future edge functions

## Quick Start

```bash
npm install
npm run dev
```

## Project Structure

- `src/`: UI pages, components, layouts, types, utils, styles
- `api/`: Edge function placeholders
- `config/`: App and integration config stubs
- `docs/`: Product and architecture documentation
- `tests/`: Unit and UI baseline tests

## Architecture Summary

- UI routes are defined in `src/App.tsx`
- Shared primitives (`Card`, `Button`, `Navbar`) live in `src/components`
- Style rules enforce white background, black text, and subtle interactions
- API files are mocked placeholders until backend integration begins
